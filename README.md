# AeroFit - Tienda virtual de ropa deportiva

Maquetacion web estatica para trabajo de clase. El proyecto representa una
tienda virtual de ropa deportiva con catalogo, filtros, carrito visual y
formulario de contacto.

## Como abrir

Abre `index.html` en el navegador. No requiere instalacion de dependencias ni
servidor local.

## Lineamientos cubiertos

- Estructura semantica en HTML5: `header`, `nav`, `main`, `section`, `article`,
  `aside` y `footer`.
- Estilos separados en `styles.css`.
- Interactividad separada en `app.js`.
- Diseno responsive para escritorio, tablet y celular.
- Maquetacion de tienda virtual: portada, beneficios, catalogo, tarjetas de
  producto, precios, filtros, carrito y formulario.
- Uso de recursos visuales locales en `assets/`.
- Navegacion interna por secciones y botones de llamada a la accion.
- Formulario con campos basicos y validacion HTML.

## Archivos principales

- `index.html`: estructura de la pagina.
- `styles.css`: apariencia visual y adaptacion responsive.
- `app.js`: filtros, carrito y formulario.
- `assets/hero-sportswear.png`: imagen principal de la tienda.
